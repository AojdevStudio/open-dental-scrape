API Employees
See API Specification

Employees GET (single)
Version Added: 23.3.22

Gets a single employee.

Parameters:

EmployeeNum: Required in URL.

Example Request:
GET /employees/17

Example Response:
{
"EmployeeNum": 17,
"LName": "Elser",
"FName": "Sierra",
"MiddleI": "O",
"IsHidden": "false",
"ClockStatus": "Working",
"PayrollID": "",
"WirelessPhone": "(555)012-4816",
"EmailWork": "ElserS@email.net",
"EmailPersonal": "",
"IsFurloughed": "false",
"ReportsTo": 0
}

200 OK
404 NotFound (with explanation)

Employees GET (multiple)
Version Added: 23.3.22

Gets a list of employees.

Parameters:

ReportsTo: Optional. FK to employee.EmployeeNum.

Example Requests:
GET /employees
GET /employees?ReportsTo=0
GET /employees?ReportsTo=17
