TaskLists GET
Version Added: 22.3.15

Gets a list of all the TaskLists filtered by TaskListStatus.

Parameters:

TaskListStatus: Optional. Either "Active" or "Archived". Default "Active".

Example Request
GET /tasklists?TaskListStatus=Archived&Offset=100
